# archana848-Nykaa-Sales-Profit-Analysis-Power-BI-Dashboard
This project analyzes sales performance, profitability, customer demographics, and product return trends in the beauty retail sector using Power BI.
The goal of this dashboard is to transform raw sales data into meaningful business insights that support data-driven decision-making.

📈 Key Insights

₹5M+ Total Sales
₹1.15M Total Profit
22.3% Profit Margin
18–25 age group driving highest revenue
Delhi as the top-performing city
10% return rate impacting profitability
